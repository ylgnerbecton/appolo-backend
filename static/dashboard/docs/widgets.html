Widgets

A widget allows you to show any statistics or other contextual information in very nice way in dashboard or other pages. In hyper widget is consisting of a title, the statistics number, relevant icon, sub statistics and sub title. You can show trends (e.g. up or down) using sub title.

                                                        var array = arg.slice();
                                                        var currentIndex = array.length, temporaryValue, randomIndex;
                                                    
                                                        while (0 !== currentIndex) {
                                                    
                                                            randomIndex = Math.floor(Math.random() * currentIndex);
                                                            currentIndex -= 1;
                                                    
                                                            temporaryValue = array[currentIndex];
                                                            array[currentIndex] = array[randomIndex];
                                                            array[randomIndex] = temporaryValue;
                                                        }
                                                    
                                                        return array;
                                                    }
