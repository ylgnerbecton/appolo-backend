Introduction

Hyper is a fully featured premium admin template built on top of awesome Bootstrap 4.1.3. It's crafted using modern web technologies HTML5, CSS3 and jQuery. The theme includes beautiful dashboard, many ready to use hand crafted components, widgets, pages, etc. The components can be used very easily on any page. The theme is fully responsive and customizable.

We really care for our buyers and so in case if you are having any question or feedback, please feel free to get back to us via email support@coderthemes.com or by filling out contact form on our website.
