Boxed Vertical Layout

If you would like to have boxes (fixed width) verticle based layout, simply add classes enlarged and boxed-layout and a data attribute data-keep-enlarged="true" to your body element and you would have it magically there.

How to change width of page?

In order to change the width, open a file src/scss/config/_custom-variables.scss and change the value of variable $boxed-layout-width. The default value is set to 1300px.
